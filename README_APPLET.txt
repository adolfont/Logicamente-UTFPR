Applet

1. Gerar o jar logicamente.jar (export, jarfiles)
2. Copiar o arquivo para o webapps\logica do tomcat
3. Criar o diretorio lib (webapps\logica\lib) e copiar a lib java_cup.jar
4. Copiar o arquivo logicamente.html para dentro do diretorio webapps\logica
5. Iniciar o servidor tomcat
6. Acessar o link http://<seu_servidor>:8080/logica/logicamente.jar

A aplicação deve ser visualizada dentro do browser
